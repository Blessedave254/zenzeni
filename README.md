# Zenzeni Africa Safari

Fast, static-first Astro website for Zenzeni Africa Safari, built with TypeScript and Tailwind CSS.

## Features

- Data-driven tour packages, destinations, activities, blog posts, testimonials, and FAQs
- SEO metadata, canonical URLs, Open Graph tags, Twitter cards, and JSON-LD schema
- TravelAgency, TouristTrip, BlogPosting, and FAQPage structured data
- Booking form UI and payment placeholders for M-Pesa STK Push and card payments
- Static sitemap at `/sitemap.xml` and robots.txt

## Commands

```bash
npm install
npm run dev
npm run build
npm run preview
```

Payment and booking API routes are placeholders for provider integration. For production POST handling, deploy with an Astro server adapter or connect forms to an external backend.
